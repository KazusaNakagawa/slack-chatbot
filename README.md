## ChatGPT Bot

### 前提
* `.env.exsample` を `.env` にし TOKEN・API 情報を入れる
* local のみ起動確認済
* Mac M1
* Python 3.9.1

### 手順

1. 実行コマンド

    ```bash
    python slack_chatbot.py
    ...

    ⚡️ Bolt app is running!
    ```

2. Slack で Botにメンションして投稿する。

3. Botからの回答を確認。
